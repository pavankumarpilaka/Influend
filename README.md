# Influend

**Influend** is a web application designed to allow users to book appointments with influencers. It is built using **React** for the frontend, powered by **Vite** for fast development and **Tailwind CSS** for styling. The application includes user authentication, a responsive layout, and an easy-to-use interface for booking appointments with influencers.

---

## Features

- **User Authentication**: Users can sign up, log in, and manage their profiles.
- **Appointment Booking**: Users can browse available influencers and schedule appointments with them.
- **Responsive Design**: The app adapts to different screen sizes to provide an optimal experience on both desktop and mobile devices.
- **Toast Notifications**: Display real-time feedback on user actions like login, sign-up, and appointment booking using **React Toastify**.

---

## Tech Stack

The application uses the following technologies:

- **Frontend**: React.js
- **Build Tool**: Vite (for fast builds and hot-reloading during development)
- **Styling**: Tailwind CSS (for utility-first styling)
- **Routing**: React Router (for navigation between different pages)
- **State Management**: React's built-in `useState` and Context API
- **Toast Notifications**: React Toastify (for success/error messages)
- **Version Control**: Git (hosted on GitHub)
- **Deployment**: Vercel (for production deployment)

## Installation

To get started with this project locally, follow these steps:

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (version 14 or higher)
- [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/) (optional)

### Clone the Repository

Clone the repository to your local machine using Git:
    git clone https://github.com/pavankumarpilaka/Influend.git
   cd Influend

Install the required dependencies by running:
npm install

Start the development server:
npm run dev

To build the project for production (minified and optimized), run:
npm run build

After building the app, you can preview the production version locally by running:
npm run preview
